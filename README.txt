There are 4 files including this README file.
1) prog1.cpp - main function
2) prog1.h - class definitions and methods
3) makefile - compiles prog1.cpp and prog1.h into prog1 executable file
4) README - this file

To compile:
1) Type “make” into command line.

To run:
1) Type “./prog1” into command line
2) Type integer value for table size. *Enter* when done.
3) Type integer value for number of keys. *Enter when done.
4) Give commands and key value as found in input.txt
5) Give command “5” to exit program.
